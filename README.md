# Bem-vindo ao **SCQT**

### Sistema para Criação de Questões de Trigonometria

O **SCQT** (Sistema para Criação de Questões de Trigonometria) é uma aplicação web desenvolvida para **auxiliar professores** na criação e organização de questões sobre **Trigonometria**.  
O sistema permite gerar **questões de múltipla escolha** e **dissertativas**, de forma rápida, prática e personalizável.

---

## Objetivo

O principal objetivo do SCQT é **automatizar e simplificar** a elaboração de listas de exercícios e bancos de questões sobre trigonometria, oferecendo uma ferramenta intuitiva e eficiente para professores e estudantes.

---

## Sobre o Projeto

O SCQT apresenta uma interface moderna e direta, desenvolvida para facilitar a interação do usuário.  
A partir da página inicial, o usuário pode compreender o propósito do sistema e navegar até a área de geração de questões, onde define a quantidade e o tipo de exercício que deseja criar.

O sistema foi pensado para abranger diversos tópicos da trigonometria, como:

- Relações métricas do triângulo retângulo  
- Relações trigonométricas em triângulos quaisquer  
- Teorema de Pitágoras  
- Seno, Cosseno e Tangente  
- Trigonometria na circunferência
