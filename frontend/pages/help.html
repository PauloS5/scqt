Como criar uma atividade?

Para se criar uma atividade neste sistema basta seguir estes passos simples:

  • Vá para página 'Questões'
  • Selecione a quantidade de questões que deseja gerar
  • Clique em 'Gerar'
  • A partir das questõies geradas escolha as categorias e estilos que cada questão
  • Ao final clique em 'Gerar Questões'
  • Confira as questões e selecione o formato de arquivo para impressão
  • Por fim, baixe e use o arquivo como quizer
